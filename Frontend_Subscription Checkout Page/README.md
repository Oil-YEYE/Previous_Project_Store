# Subscription Checkout Page
A responsive subscription checkout built with React and CSS.
This project is a frontend practice challenge focused on a clean payment layout with some selection interactions.

## Preview
[Click to view](https://snazzy-mandazi-4759cd.netlify.app/)

## Tech Stack
- React
- Javascript
- CSS
- Lucide React Icons

## Page Structure and function
- Two sections with some clickable interactions.
- Click to choose a payment method.
- Click to chose a subscription plan.
- A couple of input fields.

## Studying and Suggestion
Use nested layouts more often to uniformly adjust the interface.

# 订阅结账页面

一个使用 React 和 CSS 构建的响应式订阅结账页面。

这是一个前端练习项目，主要练习干净的支付页面布局，以及一些简单的选中交互效果。

## 预览

[点击查看](https://snazzy-mandazi-4759cd.netlify.app/)

## 技术栈

- React
- JavaScript
- CSS
- Lucide React Icons

## 页面结构和功能

- 页面分为两个主要区域，并包含一些可点击交互。
- 点击选择支付方式。
- 点击选择订阅套餐。
- 包含一些基础输入框。

## 学习记录和建议

以后可以更多使用嵌套布局，方便对界面进行统一调整。

继续加强对 Grid、display、flex 等布局方式的理解和使用。
